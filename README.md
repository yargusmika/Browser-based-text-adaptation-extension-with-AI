Расширение работающее через `Ollama`
В ollama используеться веб-версия `gpt-oss:120b-cloud`
Можно запускать на локальных образах

## Обрабатываеться "выделенный текст" и "вся страница"
Нажимаем правой кнопкой мыши, и выбираем необоходимый пункт. При выделении появляеться пункт "Упростить выделенное".
